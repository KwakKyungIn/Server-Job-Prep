## 🧤 사용 방식

1. VSCode 기반 마크다운으로 정리
2. 커밋 메시지는 `분야 + 요약` 스타일로 작성 (예: `OS: 프로세스 vs 스레드 정리`)
3. 각 문서는 다음 순서로 구성:
   - 개념 요약
   - 관련 면접 질문 & 예시 답변
   - 예제 코드 or 시나리오
   - 주의사항 및 트러블슈팅
